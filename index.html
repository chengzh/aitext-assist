🔒 隐私政策

智能文本助手 Chrome扩展

最后更新时间：2024年12月

本隐私政策描述了智能文本助手Chrome扩展如何收集、使用和保护您的信息。

📋 信息收集

我们收集的信息

  • 您选择的文本内容 - 仅当您主动选择文本并使用功能时
  • 扩展设置 - 您的API配置、语言偏好等设置
  • 使用统计 - 功能使用频率、错误日志等（匿名化）

我们不收集的信息

  • 个人身份信息（姓名、邮箱、电话等）
  • 浏览历史记录
  • 密码或敏感账户信息
  • 设备信息或位置数据

🔄 信息使用

如何使用您的信息

  • 提供核心功能 - 将您选择的文本发送给AI服务进行处理
  • 保存设置 - 在本地存储您的配置偏好
  • 改进服务 - 分析使用模式以优化功能
  • 错误诊断 - 收集错误信息以修复问题

🌐 第三方服务

AI服务提供商

我们使用以下第三方AI服务来处理您的文本：

  • Kimi (Moonshot) - 文本解释、提炼、润色、翻译
  • 阿里云百炼 - 文本处理服务
  • Claude (Anthropic) - AI文本处理

重要说明：当您使用扩展功能时，您选择的文本会被发送到这些第三方AI服务进行处理。请确保您了解并同意这些服务的隐私政策。

💾 数据存储

本地存储

  • 扩展设置存储在Chrome的本地存储中
  • 您的API密钥仅存储在本地，我们无法访问
  • 处理后的文本结果不会永久保存

云端存储

  • 我们不将您的数据存储在我们的服务器上
  • 文本处理完全依赖第三方AI服务
  • 处理完成后，文本内容不会保留

🔐 数据安全

保护措施

  • 使用HTTPS加密传输所有数据
  • API密钥仅在本地存储，不发送给我们的服务器
  • 定期更新扩展以修复安全漏洞
  • 遵循Chrome扩展安全最佳实践

📊 数据共享

我们不会分享您的信息

  • 不会向第三方出售、交易或转让您的个人信息
  • 不会与广告商或营销公司共享数据
  • 仅在法律要求时才会披露信息

例外情况

  • 当您明确同意时
  • 法律要求或政府机构要求时
  • 保护我们的权利和安全时

👤 您的权利

您有权：

  • 访问 - 查看我们收集的关于您的信息
  • 更正 - 更新或修正不准确的信息
  • 删除 - 要求删除您的数据
  • 限制 - 限制我们处理您的信息
  • 反对 - 反对我们处理您的信息

🍪 Cookie和跟踪

我们的扩展不使用Cookie或跟踪技术。我们不会跟踪您的浏览活动或在线行为。

👶 儿童隐私

我们的服务不面向13岁以下的儿童。我们不会故意收集13岁以下儿童的个人信息。如果您是父母或监护人，并且您知道您的孩子向我们提供了个人信息，请联系我们。

🔄 隐私政策更新

我们可能会不时更新本隐私政策。当我们进行重大更改时，我们会：

  • 在扩展中显示通知
  • 更新此页面的"最后更新时间"
  • 在Chrome Web Store中发布更新

建议您定期查看本隐私政策以了解任何更改。
